Avatars

Avatars are user profile pictures.

Add the avatar class to <figure> element. There are 4 sizes modifier classes available, including avatar-xl(64px), avatar-lg(48px), avatar-sm(24px), and avatar-xs(16px). By default, the avatar size is 32px.

For avatars that do not have profile pictures, you can use their initials for avatars with the default primary color as the background. Add the data-initial attribute with the users initials as the value.

Avatar Status

The avatar component support status indicators. You can add an <i> element with the avatar-status class, and use one of the following modifier classes online, busy or away for different status colors. The default is gray indicating offline.

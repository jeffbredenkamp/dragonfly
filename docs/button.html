Buttons

Easy to use button styles with support for size and other modifiers.

Examples

Primary Button Secondary Button Success Button Warning Button Error Button 
<a href="#" class="btn btn-primary" role="button">Primary Button</a>
<a href="#" class="btn btn-secondary" role="button">Secondary Button</a>
<a href="#" class="btn btn-success" role="button">Success Button</a>
<a href="#" class="btn btn-warning" role="button">Warning Button</a>
<a href="#" class="btn btn-error" role="button">Error Button</a>

Size

There are 2 sizes modifier classes available, including btn-sm and btn-lg. By default, the btn is medium and does not require an additional class.

Small Button Medium Button Large Button 
<a href="#" class="btn btn-primary btn-sm" role="button">Small Button</a>
<a href="#" class="btn btn-primary" role="button">Medium Button</a>
<a href="#" class="btn btn-primary btn-lg" role="button">Large Button</a>

Disable

Add the class disabled for disabled styles. Remember to include aria-disabled attribute.

Forms

Clean simple form styles.

Input

Basic input with labels.

<class="form-group">
  <label class="form-label" for="input-example">Email</label>
  <input type="email" class="form-input" id="input-example" placeholder="Email">
</div>

Text Box

The textarea element creates a multi-line text input.

<class="form-group">
  <label class="form-label" for="example-text-box-input">Form Label</label>
  <textarea class="form-input" id="example-text-box-input" placeholder="Text box area" rows="4"></textarea>
</div>

Select

Use select menus to combine many choices into one dropdown menu.

<class="form-group">
  <label>
    Select Menu
    <select class="form-select">
      <option>Choose an option</option>
      <option>Pound</option>
      <option>Euro</option>
      <option>Dollar</option>
    </select>
  </label>
</div>

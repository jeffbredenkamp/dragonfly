Getting Started

Dragonfly.css is a lightweight, responsive CSS framework for faster development.

Dragonfly includes basic styles for elements, responsive flexbox layout system, CSS components, utilities classes and a clean modern design.

Installation

Install manually

Download the compiled and minified Dragonfly CSS.

Download Dragonfly.css

Include dragonfly.css located in /dist in your website or Web app. 

<link rel="stylesheet" href="dragonfly.min.css">
<link rel="stylesheet" href="dragonfly-icons.min.css">

Browser Support

Dragonfly uses Autoprefixer to make most styles compatible with earlier browsers and Normalize.css for CSS resets. Dragonfly is designed for modern browsers. For best compatibility, these browsers are recommended:

Logo Vendor Browser Version
Apple Safari Latest two
Google Chrome Latest two
Microsoft Edge Latest two
Microsoft Internet Explorer 10*
Mozilla Firefox Latest two

* Dragonfly supports Internet Explorer 10+, but some HTML5 and CSS3 features are not perfectly supported by Internet Explorer.

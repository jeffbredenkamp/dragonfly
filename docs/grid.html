Example

Dragonfly's grid system uses a series of rows and columns. The below example creates 2 equal-width columns.

<div class="container">
  <div class="row">
    <div class="col-lg-6">Col 6</div>
    <div class="col-lg-6">Col 6</div>
  </div>
</div>

Gapless

Add the class gapless to the row class to have gapless columns.

6 Col Gapless
6 Col Gapless
<div class="container">
  <div class="row gapless">
    <div class="col-lg-6">Col 6</div>
    <div class="col-lg-6">Col 6</div>
  </div>
</div>

nesting

To nest with the grid, add a new row and set of columns columns within an existing column.

Col 6 Nested 
<div class="container">
  <div class="row">
    <div class="col-lg-12">
      <div class="row">
        <div class="col-lg-6">Col 6 Nested</div>
      </div>
    </div>
  </div>
</div>

Progress Bar

A progress bar displays the status of a given process.

The progress bar has 2 elements parent class progress and the value class progress-value.

<div class="progress">
  <div class="progress-value" role="progressbar" style="width:50%;" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Size

Add the class progress-sm or progress-lg on progress for different heights.

<div class="progress progress-sm">
  <div class="progress-value" role="progressbar" style="width:50%;" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"></div>
</div>

<div class="progress progress-lg">
  <div class="progress-value" role="progressbar" style="width:50%;" aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Labels

You can add text inside the progress value of a progress bar.

25%
<div class="progress">
  <div class="progress-value" style="width:25%; aria-valuenow="25" aria-valuemin="0" aria-valuemax="100">25%</div>
</div>

Colors

A progress bar can be styled with the classes .bg-color-success, .bg-color-warning and .bg-color-error.

<div class="progress">
  <div class="progress-value bg-color-success" role="progressbar" style="width:20%;" aria-valuenow="20" aria-valuemin="0" aria-valuemax="100"></div>
  <div class="progress-value bg-color-warning" role="progressbar" style="width:40%;" aria-valuenow="40" aria-valuemin="0" aria-valuemax="100"></div>
  <div class="progress-value bg-color-error" role="progressbar" style="width:60%;" aria-valuenow="60" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Multiple Bars

Include multiple progress bars in a progress component if you need.

<div class="progress">
  <div class="progress-value bg-color-success" style="width:15%;"></div>
  <div class="progress-value bg-color-warning" style="width:25%;"></div>
</div>

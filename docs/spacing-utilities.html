Spacing Utilities

Margin and padding utility classes to modify an element's appearance.

The spacing utility classes are named using the property, side and size convention. For example margin (property) top (side) and sm (size).

Examples

<!-- Margin top small -->
<div class="mt-sm"></div>

<!-- Padding right medium -->
<div class="pr-md"></div>

<!-- Padding all sides large -->
<div class="p-lg"></div>
Property
  • m
  • p
Side
  • t - set the padding or margin top only
  • b - set the padding or margin bottom only
  • r - set the padding or margin right only
  • l - set the padding or margin left only
  • x - set the padding or margin left and right only
  • y - set the padding or margin top and bottom only
  • Blank - set the padding or margin on all 4 sides.
Size
  • 0 - set size to 0
  • xxsm - set size to .375rem
  • xsm - set size to .75rem
  • sm - set size to 1.5rem
  • md - set size to 3rem
  • lg - set size to 6rem
  • xlg - set size to 12rem
  • xxlg - set size to 16rem
